## Overview

`material-absorb` is a MATLAB utility repository for plotting electromagnetic-wave absorption and material-response data. It is preserved as a focused analysis/plotting script collection.

本仓库用于保存电磁波吸收相关的 MATLAB 绘图和数据处理脚本，属于 ResearchFlow Lab 的历史科研脚本与数据可视化辅助层。

## Repository role

| Area | Purpose |
|---|---|
| MATLAB plotting | Plot absorption-related data |
| Material response | Visualize electromagnetic-wave absorption trends |
| Script archive | Preserve earlier research utilities |
| Figure preparation | Support analysis figures and manuscript visuals |
